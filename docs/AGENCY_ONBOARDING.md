# Agency Onboarding Guide

## First 30 Minutes

1. Create the agency workspace.
2. Invite internal teammates and assign production roles from `docs/ROLES_AND_PERMISSIONS.md`.
3. Add the first client.
4. Add and verify the client's primary domain.
5. Install the monitoring script or confirm crawler-only monitoring.
6. Run the first crawl.
7. Review critical issues, assign owners, and generate first AI fix briefs.

## First Week

- Import remaining clients and domains in bulk.
- Connect Google Search Console and analytics properties.
- Configure Slack or email alerts.
- Create report templates for recurring client updates.
- Schedule reports for each client.
- Review dashboard health scores daily during onboarding.

## Operating Rhythm

- Daily: review new critical alerts and failed crawls.
- Weekly: review priority issues, generated recommendations, and report drafts.
- Monthly: audit client scope, team access, billing usage, and white-label settings.

## Success Criteria

- Every active client has at least one verified domain.
- Critical pages are crawlable and monitored.
- Reports are scheduled for retained clients.
- Alerts route to the correct account owner or delivery channel.
